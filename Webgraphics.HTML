Graphic file types

Pixels
Small dots you see if you put your face too close to your television or computer screen
Resolution
Indicates the number of pixels that are displayed per inch for an image (or pixels per centimeter)
Lossy Compression
When compressed, they lose some of the image details to make the file smaller
Lossless Compression
When you compress the picture you wont compromise the picture
JPEG
Allows up to 16 million colors and is the best choice for image with many colors or color gradations (especially photographs)
PNG
Newer image format that provides a combination of many features of both jpeg and gif
SVG
XML-based vector image format for two-dimensional graphics with support for interactivity and animation
GIF
“lossless” image format, meaning that the quality of the image is not degraded through compression
